Aplicaciones necesarias

Antes de iniciar asegurate de descargar la app 'p2pcamviewer' para conectarte a los SG.

De igual modo, para grabar el video, asegurate de descargar la app 'DU Recorder'.

Paso 1: Encendido

Para encender los SG desplaza el botón lateral colocado en la tapa frontal.

Espera unos minutos a que se enciendan las cámaras.

Se encenderá una luz led de color azul cuando los SG esten listos para usarse.

Paso 2: Conexión a la red 'Q7'

Para establecer la conexión con los SG, activa el Wi-Fi en tu dispositivo.

Selecciona la red 'Q7' y escribe la contraseña '12345678'.

Paso 3: Inicia 'Plug&Play'

Una vez tu dispositivo esta enlazado a la red 'Q7', abre la aplicación 'Plug&Play'.

En el menu inferior de la aplicación selecciona la opción 'LAN'.

Este último paso mostrará una nueva pantalla 'Lista de dispositivos' donde se visualizarán las cámaras disponibles en los SG.

Paso 4: Activa la luz infrarroja

Para activar la luz infrarroja, selecciona la cámara que deseas encender de la 'Lista de dispositivos'.

Este último paso mostrará una nueva pantalla, en la cual podras controlar cada cámara en forma individual.

En el menu inferior selecciona la opción 'PTZ' para activar la luz infrarroja.

Paso 5: Grabación en tiempo real

Abre la aplicación 'DU Recorder'.

Para iniciar la grabación selecciona la opción 'Grabar'.
